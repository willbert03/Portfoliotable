## ❓about

A small webpage to share and view my graphic design portfolios on. It uses [dflip](ttps://js.dearflip.com/) to create a 3D booklet view of my work entirely through css and js. I have previously attempted to make a site like this but I did not have the capabilities until now.
<br>
If you spot any weird bugs or want to question my logic, please post a [issue](/issues/new).
